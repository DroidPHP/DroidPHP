Welcome to the DroidPHP

DroidPHP is an free and opensource standalone library developed for running PHP and mySQL based Application on your android device

This Application is totally free of charge.

This application includes LIGHTTPD, PHP (with fastcgi), MYSQL, BusyBox (only killall patches)

Your support is our driving force. Join our GitHub Project Or Fork it to help us build a better Web Server for android. Thank you!
